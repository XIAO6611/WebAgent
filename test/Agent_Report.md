# 小黄 - 宝可梦特别篇角色报告

## 基本信息
- **全名**：小黄·代·常青森林（日文：イエロー・デ・トキワグローブ，英文：Amarillo del Bosque Verde）
- **身份**：宝可梦特别篇中的主角之一，第二章的主角
- **来源**：神奇宝贝百科（https://wiki.52poke.com/wiki/小黄）

## 个性特征
### 核心特质
- **天真善良**：不愿看到任何宝可梦受伤（包括对手的宝可梦）
- **战斗风格**：非必要绝不伤害宝可梦，强调和谐共生
- **特殊能力**：
  - 与宝可梦心灵相通，可读取宝可梦心声
  - 通过宝可梦视角感知环境
  - 情绪激动时可使宝可梦等级提升
- **日常表现**：
  - 喜欢绘画，对战斗不敏感
  - 白天易困倦（因过度使用力量导致疲劳）

### 扮装原因
- 小智失踪后，小蓝为安全起见让她戴草帽掩盖性别
- 草帽内藏有小蓝的窃听装置

## 关键经历
### 红绿蓝篇
- **初次登场**：小智从发狂的迷你龙袭击中救下小黄
- **初遇宝可梦**：小智帮助她捕获第一只宝可梦——小拉达
- **重要影响**：小智"用温柔善良的心对待宝可梦"的理念及保护常青森林的行动，深刻影响了小黄

## 名字解析
- **命名来源**：中世纪欧洲命名法（名字 + de + 出身地）
- **含义**："来自常青树林的小黄"
- **类比**：类似"莱昂纳多·达·芬奇"（来自芬奇镇的莱昂纳多）
